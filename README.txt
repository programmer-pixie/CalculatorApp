This file will specify how to run the Calculator App from your device.
From a computer using the android studio: 
1. Extract the android project from the .zip file provided. 
2. Open up android studio and choose "open project"
3. Open the project extracted project. 
4. Click the green triangle "run" button.
5. You will have the option to run this project from an android virtual device,
or you can connect your own device and run the project from there.
6. Once the project is running, click the buttons to enter your equation, and click the "=" button to find the result.
